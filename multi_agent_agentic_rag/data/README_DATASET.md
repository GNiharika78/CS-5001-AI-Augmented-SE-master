# Multi-Agent Agentic RAG Sample Dataset

This dataset was created for a class project implementing the Section 4.2 multi-agent Agentic RAG workflow.

## Folders
- structured/: CSV files and SQLite database for the SQL agent
- papers/: local research/report text files for the semantic paper agent
- web/: mock recent news/policy text files for the web agent
- recommendations/: related-content metadata for the recommendation agent

## Notes
This is synthetic but realistic demo data designed to make the architecture work end-to-end.
